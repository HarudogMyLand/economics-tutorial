    货币与人民币制度¶

    章节概览¶

    • 货币

    • 人民币制度

    • 人民币制度与国际货币体系

    强大的货币是金融强国的基石，是六个强大之首。本章介绍有关货币基本要素的简单知识，为后续理解货币在流通中的作用与作用机制打下基础。

    货币¶

    货币内涵¶

    货币内涵：货币是商品交换的产物，是充当一般等价物的特殊商品1，是价值和使用价值的统一体。货币并不是贵重金属。

    重商主义认为，国家赋予的方法就是奖励出口，限制进口。这必然导致贸易量萎缩。

    货币与各个金融元素的关系：

    • 通货：一种货币形式
    • 财富：通货与存款等货币只是一种财富形式，财富还包括非货币资产
    • 收入：收入以货币形式体现。

    货币历史¶

    • 中国古代：先王制币说、交换需要说
    • 西方国家：创造发明说、交换需要说、保存财富说
    • 马克思货币观：偶然价值形式、扩大价值形式、一般价值形式与货币形式

    货币形式¶

    货币的起源大概经历了这样的阶段：

    实物货币 \(\rightarrow\) 金属货币 \(\rightarrow\) 信用货币

    实物货币，实物货币的产生需要两个条件：

    • 稀缺性：稀少且受喜爱
    • 流动性：便于转让

    缺点是形体不一，不宜分割保存，价值也不稳定。

    其次是金属货币：

    • 易分割，易保存，价值稳定
    • 铸币、称量货币

    缺点是不易携带，增长速度不能跟上经济发展。

    在中国，随着金属货币体系崩溃，信用货币体系得以建立：

    • 银行券：银行兑换的信用工具。兑现转为不兑现。
    • 不兑现纸币：流通中的纸币符号，通常由中央银行发行。

    存款货币：发挥货币职能的存款。

    现在，先进的技术使得电子货币的概念普及开来：

    • 基于金融电子化网络，通过电子技术实现的货币

    数字货币：比特币、莱特币……央行数字货币；基于区块链技术的法定数字货币。

    我国货币发展¶

    • 商：贝币
    • 春秋战国：布币、刀币
    • 秦：外圆内方
    • 宋：最早的纸币
    • 明：银本位
    • 民国：法币改革

    货币职能与作用（重点）¶

    职能¶

    1. 交易媒介：作为中介，便利交换，商品经济下交换成为必然。

    2. 价值尺度：计算衡量商品劳务的价值，为商品和劳务提供标价功能

    3. 流通手段：是一般等价物
    4. 需要现实货币，货币需求
    5. 关注货币购买力

    6. 支付手段：货币作为延期支付手段结清债权关系

    7. 价值储藏（资产职能）：作为总资产的一种存在形式，实现资产保增值。

    8. 货币可以充当价值存储手段，但不是唯一手段

    9. 货币作为价值储藏的优劣取决于物价水平。

    现代信用货币作为价值尺度为商品标价，作为流通手段完成即时交易，作为支付手段结清债权关系或价值转移，也是实现资产保值增值的一种手段

    作用¶

    1. 第一和持续推动力
    2. 媒介交易发挥促进商品发展
    3. 资产职能促进市场经济发展

    发挥作用的内在要求¶

    1. 币值稳定：
    2. 任何剧烈波动都会影响交易职能
    3. 任何剧烈波动都会影响储藏职能
    4. 货币自动调节机制：交换价值与内在价值
      1. 商品货币有自动调节机制：基于市场
      2. 信用货币需要中央银行调控货币，因为其交换价值远超内在价值。
    5. 货币数量要符合经济社会发展需求：弹性供给。这也是金属货币崩溃的原因。

    货币统计¶

    现金和各种存款货币都代表了一定的购买力，但是购买能力却不尽相同。依据流动性，IMF对货币层次有基本划分标准。

    流动性

    金融资产变现且不受损失的能力

    一些存款货币示例：

    1. 通货（Currency）：银行体系以外的现钞，居民企业持有的现钞，不包括银行的库存基金。
    2. 可转让存款（Transferable Deposits）：存款型公司发行的货期、本票、旅行支票；邮政发行可转让存款，非存款型公司发行旅行支票。
    3. 其他货币：不可转让储蓄存款，定期，固定存款。
    4. 货币市场基金份额。
    5. 债务证券:大额存单和商业票据。

    中国货币层次（重点）¶

    所有国家的货币基本上都能分为 \(M0\) 狭义货币与 \(M1\) 广义货币，在中国：

    类型 含义
    流通货币 \(M0\) = 硬币+数字人民币+现钞
    货币 \(M1\) = \(M0\) + 单位活期存款 + 个人活期 + 非银行支付机构客户背负金
    货币与准货币 \(M2\) = \(M1\) + 准货币

    准货币包括：单位定期存款、个人定期存款、证券公司客户保证金存款、住房公积金存款、其他存款、非存款机构部门持有的市场基金

    近年来，中国货币总量出现如下趋势：

    1. 货币总量快速增长：货币需求迅速增长，总供给迅速增加
    2. 货币层次结构发生变化：\(M2\)的比例迅速上升，\(M_0, M_1\) 在 \(M_2\) 中的占比不断走低

    货币制度¶

    货币制度概念（重点）¶

    货币制度就是针对货币相关要素，流通组织与管理等以国家法定使之能够发挥各种之职能的条约、法律与国际协议制度。目的是保证货币和货币流通的稳定，使之正常发挥职能。

    货币制度是伴随国家发行货币开始的，历经了金属货币制度，信用货币制度。

    本位币及其法偿能力是货币制度重要概念，一国中，基本通货称为主币或本位币，作为价格标准，本位币下小额辅助货币称为辅币。

    货币流通中，无论数额多大，何种性质，都必须接受支付的货币制度称为无限法偿；反之，超出规定范围就能够拒绝接受，为有限法偿。

    依据管辖范围不同，可分为国家货币制度，区域性货币制度，国际货币制度。

    依据货币类型不同，可分为银本位、金银复本位、金本位以及不兑现信用货币制度。

    演变¶

    • 银本位制：白银作为基本通货的本位货币，自由铸造熔化、银行券可兑换白银、银币自由出入
    • 金本位制：金币本位/金块本位/金兑换本位，制度内容主体基本和银本位制一致
    • 金银复本位：金银同时作为本位币，官价影响市价2
    • 不兑现的信用货币：货币本体几乎没有价值，货币仅仅是债务关系凭证，作为货物信用流通的工具。

    劣币驱逐良币现象，又称格雷欣法则：市场价格偏高的货币会被市场价格偏低的所排斥，导致进入储藏而劣币充斥市场

    英国政府试图使用跛行本位（低价货币不能自由铸造熔化），结果高价值货币供给未增加，而低价货币数量又受限，最终导致了通缩。

    人民币制度¶

    • “一国多币”制度
    • 保证币值稳定

    “保证流动性，但是不搞大水漫灌”解读：

    • 流动性充裕满足健康运行需要
    • 流动性合理，或多或少都不行
    • 货币政策是中性的，不能撒钱刺激经济

    国际货币体系¶

    国际货币体系：国际货币体系就是各国政府为适应国际贸易与结算需求，对货币兑换、国际收支的调节所做的安培或确定的规则，以及相应的组织形式等的总称。

    国际货币体系基本内容（重点）¶

    简而言之，可视为支配国际货币关系规则以及各国间进行各种交易支付所依据的一套安排、惯例与机构。主要职能与作用一般包括：

    • 国际储备资产规定
    • 国际间汇率敲定
    • 国际收支调节

    国际货币体系演变（重点）¶

    1. 国际金本位体系：黄金作为国际储备货币或国际本位货币的国际货币体系3，在经济大危机之后，国际金本位体系彻底崩塌

    2. 布雷顿森林体系：二战后美元为中心的国际货币体系

    3. 实行“双挂钩”、可调整汇率制，美元与黄金挂钩，其他国家汇率与美元挂钩

    4. 成立国际货币基金组织协调国际货币事务
    5. 确立了各国货币在国际收支经常账户下的自由兑换原则

    6. 各国当局有义务在外汇市场进行干预防止汇率变动幅度超越1%；4

    7. 牙买加体系：改革后，要求储备货币多元化，汇率安排多元化，调节安排多元化

    中国在国际货币组织中的责任详见金融开放一节。

    区域性货币制度与货币合作¶

    某区域内国家地区协调形成一个货币区，联合组建中央银行发行管理区域内统一货币制度。典型案例就是欧洲经济与货币联盟的单一货币体系。

    区域货币合作，指的是通过签订货币合作协议，确定参与国共同遵守的货币制度或规则，这属于较高层次的货币合作模式。、

    人民币国际化：人民币跨越国界在境外流通，成为国际上普遍认可、结算和储备货币的过程。

    1. 价值角度，具有商品性；职能角度，具有交易媒介职能。 ↩

    2. 金银复本位制度极不稳定，金银各自铸币价格不同，容易引发混乱。政府由此制定了“官价不随市价调整”或“长时间不调整”的政策。 ↩

    3. 国际金本位体系不是一个国际协定体系，单纯是基于多国金本位制度的自发货币体系 ↩

    4. 罗伯特·特里芬难题：满足国际储备需求必然导致美元收支逆差，即美国必须不断“借钱给世界”，美元信用降低，因此必须纠正美元超发，实现收支平衡；然而若要一旦停止输出美元，则会断绝国际储备来源，引起国际清偿能力不足，这就是特里芬难题。很可惜，哪怕是进入牙买加体系之后，美元长期霸占主权货币地位，使得特里芬难题长期存在。 ↩
