    金融市场构成要素¶

    金融市场¶

    金融市场是资金供求双方实现货币借贷、资金融通、办理票据和有价证券交易活动的市场。

    金融市场是金融产品发行交易的场所，是以金融产品为载体形成的货币资金供求关系及其交易机制的总和。

    • 广义上，我们的金融市场既包括直接金融、也包括间接金融。

    商品交易使用票据实现了互通有无，这是初步的商业信用，之后随着银行信用等各类信用与债权关系在错综复杂的投融资活动中健全，一个国家的金融市场逐步完全。

    金融市场要素¶

    • 参与者：政府、居民、央行、企业

    参与者按照动机角度大致可分为：

    • 筹资者
    • 投资者
    • 中介服务机构
    • 监管者

    载体：金融工具

    市场价格：在市场上，市场价格对金融工具有很大影响。

    组织形式：场内外交易。

    金融市场体系与功能¶

    金融市场划分¶

    • 按交易工具：货币市场与资本市场

    交易融通一年内资金的称为短期金融市场，即货币市场；期限一年以上的有价证券为交易工具中长期资金交易的市场是资本市场，又称中长期市场。

    • 按交割期限：现货市场与衍生品市场；
    • 按辐射地域：国内市场与国际市场

    两个特殊金融市场：黄金市场1与外汇市场

    金融市场功能与效率¶

    • 资源配置与转化功能
    • 资产流动性提供功能
    • 定价与价格发现功能
    • 风险分散和规避功能
    • 投资与财富管理功能
    • 宏观调控传导功能

    金融市场是由诸多投融资者在特定的价格机制、组织方式与交易机制下，依托市场基础设施进行金融产品发行和交易的总和，主要发挥资源配置与转化、流动性提供、定价与价格发现、风险分散和规避、投资与财富管理、宏观调控传导等功能。

    效率体现¶

    如何衡量市场效率？我们考虑市场发挥的有效效率：

    1. 金融市场活动有效性
    2. 根据有效市场假说的定价有效性
    3. 市场资源配置的有效性，即流动性高低

    建立在“公开、公平、公正”基础上的透明度是金融市场得以健康发展的制度保证。完善的价格机制、规范的市场参与者行为和活跃的金融创新是提高金融市场有效性的重要条件。

    金融市场构成¶

    金融市场体系包含多元化的金融市场体系，当前金融市场体系包含了货币市场、资本市场、衍生品市场、外汇市场、黄金市场、保险市场等。

    货币市场（重点）¶

    货币市场就是以商业银行等金融机构为主体的短期资金融通场所。如上文所述，货币市场是一个短期市场，1年以内、流动性极强、风险极低（准货币）。特点如下

    1. 交易期限短：数小时到一年不等
    2. 交易非常活跃
    3. 交易安全性高
    4. 交易额度大

    货币市场承担管理流动性、市场基准利率与调控宏观金融的作用。货币市场是流动性调剂的平台，货币市场利率在一国利率体系中具有基准性，是影响其他金融和经济指标的基础性变量。

    它是“准货币”市场。

    同业拆借市场¶

    Interbank Market，金融机构间短期资金融通的资金借贷市场，形成源于央行对商业银行法定存款准备金的要求。

    这是一种信用拆借，没有担保，基于信用进行隔夜拆借为主的短期拆借（隔夜拆借占比高达85%）。同业拆借率反映了市场上资金供给与需求力量对比。

    上海银行间同业拆借利率SHIBOR，根据18家高级银行自主报出人民币同业拆借利率，剔除最高最低的算术平均值。

    回购市场¶

    Repurchase Agreement，证券持有人在卖出一定数量证券的同时，与证券买入方签订协议，双方约定在将来某一日期由证券的出售方按约定的价格再将其出售的证券如数赎回。

    简而言之，资金融入方（卖方）在卖出证券的同时，与买方约定在未来某一日以约定价格买回该证券的交易。

    实质是以证券为抵押品而进行的一笔短期资金融通。然而回购具有完全担保品，回购利率低于同业拆借利率。

    • 质押式回购 (Pledged Repo)： 中国主流。债券的所有权不过户，只是冻结在账户里作为质押。
    • 买断式回购 (Outright Repo)： 债券的所有权发生转移（过户）。买方在持有期内可以把债券卖掉，只要到期能买回来还给人家就行（更灵活，方便做空）。

    逆回购协议（Reverse Repurchase Agreement）：证券买入方在获得证券的同时，与证券的卖方签订协议，双方约定在将来某一日期由证券的卖方按约定的价格再将其购入的证券如数卖回。

    可转让定期存单市场¶

    大额可转让定期存单市场 (Negotiable CD / NCD)，指银行发行的、可以在二级市场上流通转让的定期存款凭证市场。

    与普通定期存单的区别（对比考点）：

    • 只有下限金额，面额大： 非标准化产品，机构投资者为主
    • 不记名、可转让： 流动性极强（普通存单没到期取出来要罚息，NCD直接卖掉就行）。
    • 利率市场化： 利率通常高于同期存款。

    中国目前流行同业存单，是中小银行补充流动性、进行主动负债管理的重要工具。

    大额可转让同业存单：发行利率、价格等以市场化方式内确定，一般按浮动利率或固定利率计息，并参考SHIBOR定价。

    银行承兑票据与短期融资券市场¶

    企业市场化融资和流动性管理的平台。

    • 银行承兑票据市场：这是由承兑申请人（企业）开出，并由银行审批并承诺到期无条件付款的票据

    • 短期融资券市场：这是具有法人资格的非金融企业在银行间债券市场发行的，约定在1年内还本付息的有价证券

    短期政府债券和央行票据市场¶

    短期国债通常是为了融通短期资金，调节财政季节性头寸不足，达到年度财政平衡；也可作为调整经济的财政工具手段；央行票据是中央银行（中国人民银行）发行的短期债务凭证，曾作为货币政策的一种调节货币市场流动性。

    核心特征：

    • 违约风险极低： 被称为无风险资产（Risk-free Asset）。
    • 流动性极强： 随时能变现，近似于现金。
    • 发行方式： 通常采用贴现发行（低于面值卖，到期按面值还，差价就是利息）

    是市场无风险基准利率的天然锚，央行进行公开市场操作的主要买卖对象。

    资本市场（重点）¶

    广义包括长期借贷市场和证券市场，侠义是指证券市场。

    资本市场通过投资融资、发现价格、管理风险、并购重组以及激励创新，促进资源的有效配置和产业结构的优化升级。

    股票市场¶

    股票是股份公司发行的所有权凭证，可转让、买卖、抵押

    股票可分为普通股和优先股。

    发行制度：审批制、核准制、注册制

    股票交易市场：

    股票交易场所：场内交易与场外交易

    股票交易制度：竞价交易制度、做市商交易制度

    债券市场¶

    债券发行方式：承销发行和招标发行

    现券交易、回购交易、衍生品交易

    衍生品市场（重点）¶

    衍生品(Derivatives)是指在一定的原生工具或基础性工具之上派生出来的金融工具，其形式是载明买卖双方交易品种、价格、数量、交割时间和地点等内容的规范化或标准化合约与证券。

    衍生品具有跨期性、杠杆性、高风险性和合约存续的短期性等特征。

    常见的衍生工具种类：期权、期货、远期、互换等。

    衍生工具的标的物：实物、基础证券、衍生工具、其它。
