这门课是基于 MIT14.02，和北京航空航天大学央财联合培养宏观经济学课程的笔记。

宏观经济学，观察一片广阔森林的气候与生态，它关注的不是单棵树木的形态，而是森林整体的生长周期、物种消长与物质循环。与之相对，微观经济学则聚焦于个体决策——家庭、企业如何在市场中互动。两者视角不同，共同构成了我们理解经济现象的全景图。宏观经济学不仅仅是微观的简单叠加，它研究从微观角度更复杂的事物。

宏观经济学的研究，始于那些关乎国计民生的核心议题：

- **增长与波动**：为何有些国家富裕而有些国家长期贫困？经济为何会经历繁荣与衰退的周期？
- **失业与劳动市场**：我们如何准确诊断就业市场的健康？例如，失业率这一指标背后，隐藏着怎样的结构性变化与个体抉择（如劳动参与率的波动）？
- **价格与货币**：通货膨胀或通货紧缩的根源是什么？中央银行如何通过货币政策影响整体经济？
- **开放经济的互动**：汇率如何决定？国际贸易与资本流动如何将不同国家的命运紧密相连？

整体经济会呈现出个体所不具备的、复杂的“涌现”特性。个人的节俭，若成为全社会的普遍行为，却可能导致总需求不足与衰退——这就是经典的“节约悖论”。因此，宏观经济学发展出自己独特的分析框架与工具，如国民收入账户、总供给与总需求模型等，以研究这些从微观视角无法直接推导的宏观现象。

作为一门原理课程，我们将**着眼于核心经济思想的逻辑与直觉**，非复杂的数学推导。学习如何使用这些基本框架，来解读现实世界的经济、政策辩论与长期趋势。
