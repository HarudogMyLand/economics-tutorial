

了解宏观经济学的研究对象是国民经济总体，关注的是经济增长、失业、通货膨胀等总量问题。但要真正“看懂”宏观经济，我们首先需要宏观经济学的基本概念和衡量指标。这些工具就是国民收入核算中的核心概念。

下面我们逐一介绍。

### 总产出

衡量经济总产出最常用的指标是国内生产总值，简称GDP。GDP衡量的是一个国家在特定时期内（通常为一年）所生产的全部最终产品和服务的市场价值。需要特别注意的是，GDP不仅包括汽车、家电等实物商品，也包括教育、医疗、金融等服务的价值。

要理解GDP，我们可以从三个角度来观察：**支出角度**、**生产角度**和**收入角度**。这三个角度实际上是同一事物的不同侧面，理解它们之间的关系是掌握宏观经济学的基础。

**支出法**：是从最终产品的购买者角度来核算GDP。我们把全社会对最终产品的购买支出分为四个部分：消费（C）、投资（I）、政府购买（G）和净出口（X-M）。用公式表示为：GDP = C + I + G + (X - M)。其中净出口等于出口减去进口，因为进口产品并非本国生产，需要从总支出中扣除。

**生产法**（也称为增值法）：是从生产过程中价值增加的角度来核算。每一个生产环节创造的价值，等于该环节产出价值减去所消耗的中间产品价值。将经济中所有生产部门的增加值加总，就得到GDP。这种方法可以有效避免重复计算。

**收入法**：则是从生产要素获得收入的角度来核算。生产过程中，劳动获得工资，资本获得利息，土地获得租金，企业家才能获得利润。将这些要素收入加总，再加上间接税和折旧，就得到GDP。

这里有一个贯穿宏观经济学的重要原理：**总支出等于总产出等于总收入**。为什么这三个量会相等？因为每一笔交易中，买者的支出就是卖者的收入，而卖者收入的总和又等于生产出来的产品价值。这个恒等式不仅是国民收入核算的基础，也是后续分析国民收入决定的理论起点。

### GDP与GNP

在阅读经济新闻时，我们还会遇到另一个指标——国民生产总值，简称GNP。GDP和GNP虽然只有一个字母之差，但核算原则完全不同。

GDP是**地域概念**，它衡量的是在一国境内生产的最终产品价值，无论生产者是本国企业还是外国企业。例如，日本丰田公司在美国工厂生产的汽车，计入美国的GDP。

GNP是**国民概念**，它衡量的是由一国国民拥有的生产要素所生产的最终产品价值，无论生产活动发生在国内还是国外。同样以丰田为例，其美国工厂生产的汽车计入日本的GNP。

两者的关系可以用公式表示：$$
GNP = GDP + 本国要素从国外获得的收入 - 外国要素从本国获得的收入$$
当一个国家有大量企业在海外经营时，GDP和GNP的差异可能相当显著。

### 名义GDP与实际GDP

GDP的数值会随着物价水平的变化而变化，而物价水平与实际产出未必严格相关。如果所有产品的价格翻倍，即使实际生产的数量没有变化，GDP也会翻倍。为了区分价格变动和数量变动对GDP的影响，经济学家区分了名义GDP和实际GDP。

**名义GDP**是用当年价格计算的GDP，也就是我们通常从统计公报中看到的数值。**实际GDP**则是用某一年（称为基年）的不变价格计算的GDP，它剔除了价格变动的影响，只反映实际生产数量的变化。

用符号表示，记名义GDP为$\$Y_t$（带个美元符号），实际GDP为$Y_t$。两者之间的关系由GDP平减指数联系：GDP平减指数 = 名义GDP / 实际GDP。这个指数反映了经济中总体价格水平相对于基年的变化。显然在基年，名义GDP等于实际GDP，GDP平减指数为100。

实际GDP的增长速度，才是我们通常所说的“经济增长率”，它真正反映了一国生产能力的扩张。

### 失业率

除了总产出，失业率是衡量经济健康状况的另一个核心指标。要理解失业率，首先需要明确什么是“失业”。

并非所有没有工作的人都算失业者。一个大学生毕业后尚未找到工作，如果他正在积极求职，就算失业；但如果他已经放弃找工作，则不被计入失业人口。因此，失业者的定义是：**没有工作、正在积极寻找工作、并且能够立即上岗的人**。

劳动力是就业人口与失业人口之和。失业率的计算公式为：

$$失业率 = 失业人口 / 劳动力 × 100\%$$
$$u=\frac{U}{L}$$

这里还需要引入一个相关概念——**劳动参与率**。劳动参与率衡量的是适龄劳动人口中有多少人在积极工作或找工作，计算公式为：
$$劳动参与率 = 劳动力 / 劳动年龄人口 × 100\%$$
$$劳动参与率 = \frac{E + U}{L} × 100\%$$

劳动参与率和失业率共同反映了劳动力市场的状况。例如，如果失业率下降但劳动参与率也在下降，可能意味着更多人退出了劳动力市场，而不仅仅是就业状况改善。美国的劳动参与率在2020年新冠疫情后一度大幅下降，虽然后来失业率回升，但劳动参与率仍未完全恢复，这表明一部分人尚未回到劳动力市场。

从经验规律来看，实际GDP增长越快，失业率往往越低。这种反向关系是理解经济周期的重要线索。

### 通货膨胀率

通货膨胀率衡量的是物价水平持续上涨的速度。经济学家常用两种指标来度量通货膨胀：GDP平减指数和消费者价格指数（CPI）。

**GDP平减指数**是名义GDP与实际GDP的比率，它衡量的是经济中所有最终产品和服务的价格水平。第t期的通货膨胀率可以用GDP平减指数计算：

$$π_t = (P_t - P_{t-1}) / P_{t-1}$$
其中 $P_t$ 是第t期的GDP平减指数。

消费者价格指数（CPI）则衡量的是典型城市家庭购买的一篮子消费品和服务的价格变化。与GDP平减指数不同，CPI只包含消费品（不包括资本品和政府产品），并且包含进口消费品。此外，CPI使用固定的产品篮子，而GDP平减指数随着每年产出结构的变化而变化。

这两个指标各有用途。GDP平减指数更全面地反映整个经济的价格水平，而CPI更直接地反映消费者面临的生活成本变化。在实际政策制定中，中央银行通常更关注CPI，因为它直接关系到居民的实际购买力。
