# 微观经济学-市场理论 III

在前两章，我们考察了市场的两个极端：完全竞争（原子式的众多小厂商）和完全垄断（一个厂商独占市场）。然而，现实世界中的大多数行业，比如汽车、航空、电信、电商平台，既不是原子式的，也不是独家垄断的。它们通常由几家规模庞大的企业主导，这些企业之间存在激烈的竞争，同时又深刻地相互依赖。这种市场结构，就是**寡头垄断**。

当市场上只有少数几个卖家时，任何一个企业的决策——比如是否降价、是否增产——都会直接影响其他企业的利润，并可能引发对手的强烈反应。因此，分析寡头市场，我们不能再像之前那样，把厂商视为孤立的决策者。我们必须引入一种全新的分析工具，来研究这种**策略性互动**。这个工具就是**博弈论**。

#### 博弈论入门：策略与均衡

博弈论是研究理性决策者在相互依存的情况下如何做出选择的理论。一个博弈包含几个基本要素：参与者、他们的**策略**、以及博弈结束后的**收益**。

**纳什均衡：一个关键的均衡概念**
博弈论中最核心的均衡概念是**纳什均衡**。它描述了一种状态：在给定其他参与者策略的情况下，没有任何一个参与者可以通过单方面改变自己的策略来获得更高的收益。简单说，就是每个人都已选择了对对手策略的最优反应，没有人想“出轨”。

最经典的博弈论案例是**囚徒困境**。两个囚徒被分开审讯，他们可以选择“坦白”或“沉默”。收益矩阵如下（数字代表刑期，越小越好）：

-   如果双方都沉默，各判1年。
-   如果A坦白、B沉默，A获释（0年），B判5年。
-   如果B坦白、A沉默，B获释，A判5年。
-   如果双方都坦白，各判3年。

在这个博弈中，无论对手选择什么，选择“坦白”总是能给囚徒带来更短的刑期（如果对方沉默，坦白比沉默好：0 < 1；如果对方坦白，坦白也比沉默好：3 < 5）。因此，“坦白”是一个**占优策略**。最终，理性的双方都会选择坦白，达成纳什均衡 (3, 3)。但这个结果对双方而言，显然比相互合作、都选择沉默的 (1, 1) 要差。囚徒困境揭示了个人理性与集体理性之间的矛盾，它在解释寡头之间的合谋与背叛时至关重要。

#### 经典寡头模型：产量竞争 vs. 价格竞争

有了博弈论的思想，我们可以开始分析几种经典的寡头竞争模型。它们的基本设定都是**双寡头**，即市场上只有两个厂商：厂商1和厂商2。

**1. 古诺模型（同时定产）**
古诺模型是最早的寡头模型。它的核心思想是：两家厂商生产同质产品，**同时决定各自的产量**，然后市场根据总产量决定一个统一的价格。

-   **假设**：市场需求曲线为线性 $P = a - bQ$（其中 $Q = q_1 + q_2$），两家厂商的边际成本相同且恒为 $c$。
-   **决策逻辑**：每家厂商在决定自己产量时，都把对手的产量当作是固定的。厂商1的利润函数为 $\pi_1 = [a - b(q_1 + q_2)]q_1 - c q_1$。对 $q_1$ 求导并令其为零，我们可以得到厂商1对厂商2的**反应函数**：$q_1 = R_1(q_2) = (a - c - b q_2)/(2b)$。同理，可得厂商2的反应函数 $q_2 = R_2(q_1)$。
-   **古诺均衡**：两个反应函数的交点，就是古诺均衡。此时，每家厂商的产量都是对对手产量的最优反应，没有任何一方有单方面改变的动机。这正是一个纳什均衡。解联立方程可得：
    $$
    q_1^* = q_2^* = \frac{a-c}{3b}, \quad Q^* = \frac{2(a-c)}{3b}, \quad P^* = \frac{a+2c}{3}
    $$

**2. 斯塔克伯格模型（先后定产）**
与古诺模型不同，斯塔克伯格模型引入了一个**先动优势**。假设厂商1是**领导者**，先决定产量；厂商2是**追随者**，在看到厂商1的产量后再决定自己的产量。这是一个**序贯博弈**。

-   **决策逻辑**：我们需要用**逆向归纳法**来求解。首先分析追随者厂商2。它在给定领导者产量 $q_1$ 的情况下做决策，其反应函数和古诺模型中一样：$q_2 = (a-c-bq_1)/(2b)$。
-   领导者厂商1是聪明的，它**预判**到了追随者的反应。因此，它会把 $q_2$ 的这个反应函数代入自己的利润函数中：$\pi_1 = [a - b(q_1 + \frac{a-c-bq_1}{2b})]q_1 - c q_1$。
-   **斯塔克伯格均衡**：领导者厂商1在这个基础上选择最大化自己利润的产量。求导可得：
    $$
    q_1^* = \frac{a-c}{2b}, \quad q_2^* = \frac{a-c}{4b}
    $$
    可以看到，领导者获得了更大的市场份额和利润，体现了先动优势。

**3. 伯特兰模型（同时定价）**
古诺和斯塔克伯格都假设厂商以**产量**为决策变量。但法国经济学家伯特兰提出，厂商实际上是以**价格**为决策变量的。这个看似微小的变化，会导出完全不同的结论。

-   **假设**：两家厂商生产**同质产品**，**同时决定各自的价格**。消费者完全知情，并且只会从价格最低的厂商那里购买。如果价格相同，则平分市场。假设两家厂商的边际成本同为 $c$。
-   **伯特兰悖论**：想象一下，如果厂商1定价高于 $c$，厂商2就有动机将价格定得比厂商1略低一点点，从而抢走全部市场。但厂商1又会反过来把价格定得比厂商2更低。这种“价格战”会一直持续下去，直到价格降到等于边际成本 $c$ 为止。此时，任何一家厂商都无法再通过降价获利（因为会亏本），也无法提价（因为会失去所有顾客）。
-   **伯特兰均衡**：最终，均衡价格为 $P_1 = P_2 = c$，两家厂商获得零经济利润。这个结果与完全竞争市场的结果一模一样！只需要两家厂商进行价格竞争，就能达到完全竞争的效率。这被称为伯特兰悖论。

如果产品是**差异化**的，情况就不同了。此时，即使一家厂商定价稍高，一些忠实于其品牌的消费者也不会全部流失。需求函数可以写为 $q_1 = a - p_1 + d p_2$。这样，每家厂商面临的都是一条向下倾斜的需求曲线，均衡价格将高于边际成本，利润为正。

#### 深入博弈论

寡头模型本身就是博弈论的具体应用。现在，我们抽象地梳理一下博弈论的几个关键拓展，它们能帮助我们理解更复杂的策略互动。

**1. 静态博弈与混合策略**
在有些博弈中，不存在像囚徒困境那样的纯策略纳什均衡。比如“猫抓老鼠”游戏，一方的得益正好是另一方的损失。此时，参与者可能会随机化自己的选择，以让对手无法捉摸。这种以一定概率随机选择策略的方式，称为**混合策略**。其均衡的求解逻辑是：调整自己的概率分布，使得对手在选择任何纯策略时的期望收益都相等，即让对手无差异。

**2. 动态博弈与可置信威胁**
在斯塔克伯格这样的序贯博弈中，后行动者可以看到先行动者的选择。分析这类博弈的标准方法是**逆向归纳法**：从博弈的最后阶段开始，分析最优选择，然后倒推回前面阶段。这个方法的核心是剔除**不可置信的威胁**。例如，在一个位势厂商可能会威胁新进入者：“如果你进入，我就发动价格战。”但如果发动价格战会让位势厂商自己损失惨重，这个威胁就是不可置信的。逆向归纳法会自动剔除这样的威胁，得到的均衡称为**子博弈完美纳什均衡**。

**3. 重复博弈与合作的涌现**
在一次性囚徒困境中，合作无法达成。但如果博弈被**无限次重复**呢？此时，情况可能会改变。因为参与者会考虑到未来的收益，愿意用今天的合作换取明天的持续合作。一种著名的策略是**冷酷策略**：我首先选择合作，只要你一直合作，我就一直合作；但如果你有一次背叛，我从那以后永远选择背叛。只要参与者足够有耐心（即未来的收益贴现因子足够大），这种“以牙还牙”的机制可以维持合作，让囚徒困境走出“双输”的结局。这为解释现实中的寡头合谋（如OPEC）提供了理论基础。

从产量竞争到价格竞争，从静态博弈到动态博弈，寡头理论为我们揭示了一个复杂而迷人的策略互动世界。在这里，企业的命运不仅取决于自己的决策，更深刻地依赖于对手的反应。这正是现实市场中竞争与合作的真实写照。
