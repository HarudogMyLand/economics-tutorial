# 远期期货概述

## Cheat Sheet

多头：将来买入

空头：将来卖出

1. 金融远期：双方约定在未来某一确定时间，按照约定价格买卖一定数量某种数量合约
2. 金融期货：在交易所交易、协议双方按约定在将来某个时期按照确定条件买入卖出一定价格数量的特定金融工具的标准化协议

区别对比：

| 区别         | 远期             | 期货                     |
| ------------ | ---------------- | ------------------------ |
| 交易场所     | 无固定场所       | 交易所集中交易           |
| 标准化       | “契约自由”       | 标准化条款               |
| 违约风险     | 取决于信用       | 交易所或清算机构担保     |
| 双方关系     | 直接签订合约     | 交易所或清算机构报价     |
| 价格确定方式 | 直接判断私下决定 | 交易所公开竞价和做市报价 |
| 结算方式     | 到期结算         | 每天结算                 |
| 结清方式     | 实物或现金交割   | 到期交割或平仓结清       |

远期合约种类

1. 远期利率协议
2. 远期外汇协议
3. 远期股票合约：规避系统风险

期货头寸结清：买入仓，卖出仓，到期交割
